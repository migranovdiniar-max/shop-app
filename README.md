# Shop SPA (Laravel + Vue)

Одностраничное приложение с каталогом товаров, аутентификацией и интерактивным UI
на одной странице (modals + drawer).

## Stack
- Frontend: Vue 3, Pinia
- Backend: Laravel API
- DB: MariaDB
- Cache: Redis
- Auth: Laravel Sanctum
- UI: SPA without page reloads

## Features (WIP)
- Public product catalog
- Auth (login / register / logout)
- Favorites & Cart (authorized users)
- Modal-based navigation (no extra pages)
